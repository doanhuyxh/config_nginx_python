📋 Nginx Config Manager

API để quản lý cấu hình Nginx cho nhiều domain trên các hệ điều hành Linux khác nhau (Ubuntu, Debian, CentOS, RHEL, Fedora). Hỗ trợ deploy một domain hoặc nhiều domains cùng lúc, tự động cấu hình reverse proxy và SSL.

🚀 Hướng dẫn nhanh

1. Deploy một domain:

curl -X POST http://localhost:3000/deploy/nginx \
  -H "Content-Type: application/json" \
  -d '{
    "domain": "example.com",
    "app_url": "http://localhost:3000",
    "ssl": "yes"
  }'

2. Deploy nhiều domains cùng lúc:

curl -X POST http://localhost:3000/deploy/nginx/batch \
  -H "Content-Type: application/json" \
  -d '{
    "domains": [
      {
        "domain": "example1.com",
        "app_url": "http://localhost:3000",
        "ssl": "yes"
      },
      {
        "domain": "example2.com",
        "app_url": "http://localhost:3001",
        "ssl": "no"
      }
    ]
  }'

⚠️ Lưu ý: Cần chạy với quyền root (sudo)

ℹ️ Thông tin hệ thống

  • Ubuntu/Debian: Config files stored in /etc/nginx/sites-available/
  • CentOS/RHEL/Fedora: Config files stored in /etc/nginx/conf.d/
  • SSL: Tự động cấu hình SSL với Certbot (LetsEncrypt)
  • Reverse Proxy: Tự động cấu hình proxy đến backend app
